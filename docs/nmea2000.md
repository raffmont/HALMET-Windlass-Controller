# NMEA 2000

The firmware sends NMEA 2000 windlass telemetry directly with the
ttlappalainen NMEA2000 library and the `NMEA2000_twai` ESP32 transport.

## HALMET CAN Setup

The local HALMET constants are used:

```text
kCANTxPin = GPIO19
kCANRxPin = GPIO18
```

The node is configured as `N2km_NodeOnly`, publishes product/device information,
opens the CAN interface with `Open()`, and calls `ParseMessages()` from a
1 ms SensESP event-loop callback.

## Transmitted PGNs

| PGN | Name |
| ---: | --- |
| `128776` | Windlass Control Status |
| `128777` | Anchor Windlass Operating Status |
| `128778` | Anchor Windlass Monitoring Status |

## Application Fields

The outgoing messages include the local device instance, active command, manual
UP/DOWN state, deployed length, speed, pulse count, and fault bitmask where
those fields are packed by the firmware.

NMEA field definitions are licensed by NMEA. Validate byte-level interpretation
with the vessel MFD, a certified NMEA 2000 analyzer, or the official database
used by the integrator before relying on displayed values operationally.
